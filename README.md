# Nexus Core AI - Identidade Visual e Landing Page

Este repositório contém o código-fonte da landing page oficial da **Nexus Core AI**.

## 🚀 O Diferencial
A Nexus Core AI não é apenas um chatbot. É uma plataforma de **Inteligência de Mercado** e **Automação de Vendas** focada em:
- **Imobiliárias:** Atendimento 24/7 e qualificação por perfil de crédito.
- **Concessionárias:** Triagem automática de interesse em modelos e consórcios.

## 🛠️ Tecnologias Utilizadas
- **HTML5 & CSS3 Premium:** Design com glassmorphism e animações fluidas.
- **Vanilla JS:** Interações de ultra-baixa latência (sem dependências pesadas como LangChain no front-end).
- **SEO Optimized:** Tags prontas para indexação e alta performance.

## 🎨 Identidade Visual
- **Cores:** Midnight Blue, Electric Cyan e Metallic Slate.
- **Tipografia:** Outfit (Títulos) e Inter (Sistema).
- **Logo:** Hexágono tecnológico inspirado em núcleos de processamento de dados.
